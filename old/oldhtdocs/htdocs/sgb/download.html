Install Steps:

0) Install the Ogre SDK for Visual Studio 2005 (Full or Express) here

1) Unzip this source into your desired directory

2) Open up the Visual Studio 2005 solution file provided.

3) Set the working directory by doing the following:
- Go to Project -> SGB Properties
- Select "Debugging" under "Configuration Properties"
- Set "working directory" to "bin\$(ConfigurationName)"

4) Compile and Run the project in Debug mode.

Gameplay:

Try to get your character to the end of the level and jump on the Testudo statue to win! Jump on the bad guys to vanquish them!

Controls:
Left/Right Arrow Keys - Walk
Shift Key - Hold while walking to run
Space Bar - Jump
Esc - Pause Menu (Here you can choose to restart the level or quit)
M - Brings up the Menu. Use the Menu to select the Model and Level to load

Power Ups:
Clocks - Project Extention! Add Time to your timer
Mountain Dew - "Do The Dew!" Limited Time Invincibility
A+ Project - Ego Boost! Grow Big, Jump Higher, and allow yourself another hit before dieing
