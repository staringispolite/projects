Automatically adds Google +1 buttons to your posts and lets you customize it's display methods

== Description ==

Automatically adds Google +1 buttons to your posts. It works for single pages and loops.
The plugin features an options page in which the following can be configured:

Display options:

* Toggle hiding of +1 buttons in loops 

Display style:

* Preview of the button
* Show post on top or bottom of post
* Add breaks before and/or after the button
* Button size (small,standard,medium,tall)
* Toggle showing of counter
* Choose between bubble and inline counter display (**New since 1.5**)
* Custom styling option, adds a containing div with classname which you can style in your stylesheet

Advanced options:

* Set the language of the mouseover text
* Set parse option (onload,explicit)
* Set JS Callback function
* Toggle Asynchronous mode (**New since 1.4**)
* Toggle HTML5 valid syntax (**New since 1.4**)

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and move the `google-plusone` folder to your `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. After activation you can configure the plugin using the `settings` link or the menu
