== Description ==
 
Make your twitter feed appear of your wordpress blog. <br>
Note: The client must have internet access for this plugin to work - does not require server to have internet access.

== Installation ==

<h3>Install</h3>

1. Activate Plugin.<br>

2. Go to Settings>> In Twitter<br>

3. Enter your Username, height(standard height is 211) and how many twitter feeds you want displayed.<br>

4.  Press "Update" <br>

5. Go to Appearance >> Widgets >> and place a "Text" box in. Name the Text Box twitter (or whatever you want)<br>

6. Place the following in the "text area"<br>

<code>&lt;div id='twitterbox' class='twitterbox'>&lt;/div></code>

<h3>To Uninstall</h3>

1. Deactivae the plugin
2. Remove the textbox widget
